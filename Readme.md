```shell
├── fonts/               # шрифты
├── images/              # изображения

├── .editorconfig        # конфиг для редакторов
└── [...]                # other...
```

## Динамическая перезагрузка браузера

    - Убедиться что установлен node.js (node -v) (https://nodejs.org/en/)
    - Устанавливаем browsersync (https://browsersync.io/)
    - Запускаем: npm run start
